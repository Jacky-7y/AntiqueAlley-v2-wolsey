# Antique Alley | Kingston, ON

A modern, high-performance migration of the Antique Alley website. This project transitions the original static HTML/CSS/JS site into a Single Page Application (SPA) using React and Tailwind CSS v4.

1. Install Necessary Software

    Download Node.js: Go to nodejs.org and download the LTS version. Run the installer and click "Next" until finished.

    Download VS Code: Go to code.visualstudio.com and install it.

2. Prepare the Project Folder

    Open VS Code.

    Press Ctrl + ` (the backtick key next to the number 1) to open the internal terminal.

    Type cd Desktop and press Enter.

    Type git clone https://github.com/Jacky-7y/AntiqueAlley-v2-wolsey.git and press Enter.

    Type cd AntiqueAlley-v2-wolsey and press Enter.

3. Install the Website Components

    In the same terminal window, type npm install and press Enter.

    Wait for the progress bar to finish. This downloads all the "gears and wires" needed for the site to run.

4. Start the Website

    Type npm run dev and press Enter.

    Look for a link that looks like http://localhost:5173.

    Hold Ctrl and Click that link, or copy and paste it into your web browser.

5. To Stop the Site

    Go back to the terminal in VS Code.

    Press Ctrl + C on your keyboard.

    Type y and press Enter to turn off the server.
